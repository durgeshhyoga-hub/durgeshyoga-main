# Website Content Editing Guide

All text content on the website is now fully editable through the **Admin Panel > Edit Content** page. This guide explains which fields control which parts of the website.

## 1. Header & Hero Section
- **Header**: Edit navigation labels and the "Book a Session" CTA button text.
- **Hero**: Update the main title, subtitle, badge text, and both primary/secondary buttons.
- **Stats**: Update the counts and labels for "Sessions Conducted", "Participants Trained", etc.

## 2. About Section
- **Main Content**: Update the section title, two description paragraphs, and the "govt. certified" floating card text.
- **Highlights**: Edit the 4 key highlight points (icon text) displayed below the description.

## 3. Services Section
- **Services 1-5**: For each service, you can edit:
    - **Title**: The main name of the service.
    - **Description**: A short paragraph describing it.
    - **Features**: A comma-separated list of bullet points (e.g., "Feature 1, Feature 2, Feature 3").

## 4. Experience Section
- **Experience 1-5**: Manage up to 5 experience entries.
    - **Institution**: Name of the organization.
    - **Program**: Name of the session/program delivered.
    - **Type**: Category (e.g., "Healthcare", "Education").
    - **Description**: Details about the engagement.

## 5. Skills Section
- **Skills 1-5**: Manage up to 5 key skills.
    - **Title**: Name of the skill.
    - **Description**: Brief explanation.
    - **Level**: Proficiency percentage (0-100).

## 6. Gallery & Contact
- **Gallery**: Update the section title and description.
- **Contact**:
    - **Header**: Section title and intro text.
    - **Info**: Sidebar title and description.
    - **Details**: Phone, Email, Instagram handle/URL, and Location text.

## 7. Footer
- **Description**: The short bio displayed in the footer.
- **Copyright**: The copyright text at the very bottom.

> [!TIP]
> Changes are saved immediately to the database but may take a moment to reflect on the live site depending on caching. If you don't see changes, try refreshing the page.
